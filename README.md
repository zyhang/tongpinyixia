## 一句话

在抖音信息流中，用一道没有标准答案的轻问题，让用户看见真实群体比例，并读到一条经授权、已审核的同选理由，获得一次短暂、安全、无社交压力的共鸣。

## 核心决策（Phase 1 MVP）

- **保留**：A/B 轻问题、真实比例、授权理由池、共鸣反馈、回声卡分享、A/B 实验
- **删除**：兴趣重合度、随机百分比分数、人格标签、实时匹配
- **延后**：用户投稿、开放文本、实时撮合、跨用户关系能力

## 发布门禁

本产品按阶段推进，每阶段验收通过后才进入下一阶段：

1. **Phase 0** ← 当前 — 能力确认与素材准备
2. **Phase 1A** — 内部联调与小样本基线
3. **Phase 1B** — 最小 A/B 实验
4. **Phase 2** — 受控投稿（仅 Phase 1 成立后评估）
5. **Phase 3** — 真实匹配评估（单独立项）
