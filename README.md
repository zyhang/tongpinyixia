## 一句话

选一个更像你的答案，看看有多少人和你一样，再听听他们为什么这样选。

## 当前 MVP

- **主循环**：今日一问 → 选择答案 → 看真实比例与真实理由 → 再看一道题
- **可选支线**：从结果页写下一句匿名理由，再回到下一题
- **保留**：A/B 轻问题、真实比例、授权理由池、轻反馈、受控匿名理由投稿
- **不做**：分享卡、保存图片、实时匹配、关系建立、个人投稿记录、撤回入口、人格标签

## 当前设计基线

- 页面结构和交互路径以 `designs/wireframes/` 为准，当前视觉实现以 `designs/high-fidelity/` 为准。
- 旧高保真预览已经移除，不再作为设计或验收依据。
- 新高保真与当前 MVP 主循环逐页对应；后续改动需同时更新线框图与高保真基线。
- 历史 PRD 与当前准则冲突时，以《产品理念与 MVP 设计准则》和《答案之外｜品牌文案规范》为准。
