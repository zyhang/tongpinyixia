# 好题标准｜同频一下题库设计准则

> 文档状态：定稿  
> 依据：PRD-同频一下.md + 团队讨论（2026-07）+ 多轮题库迭代复盘  
> 用途：题库设计、评审、扩题时的共同判断框架  
> 优先级：本文件中的规则均为硬性约束，以"必须"、"禁止"、"不得"开头的条目为强制执行项

---

## 核心原则

好的题目不是"有道理"的题目，而是**用户在抖音信息流中刷到时愿意停下来点一下**的题目。

以下六条标准是必选项，不是加分项。新题若有一条不满足，直接淘汰。

---

## 标准一：一秒题感

用户扫一眼就该知道这题在问什么、两个选项各自代表什么。不需要读第二遍，不需要断句。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 周末早上醒来你会 | 周末你更倾向于选择哪种状态度过早晨时光 |
| 吃火锅你更在意 | 在火锅用餐体验中你最看重的因素是什么 |

**判断方法**：一个初中生水平的人，在 0.5 秒内能说出这题是问什么的。

---

## 标准二：自己有答案

用户在读到选项之前，心里已经有了答案。这题不是在帮用户"做决策"，而是在让用户**确认自己已有的偏好**。

如果用户需要权衡利弊、回忆数据、查资料才能回答，就不会在信息流里停下来。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 吃西瓜你更喜欢 → 你立刻知道 | 你认为哪种投资方式更适合年轻人 → 需要想 |
| 收到消息你习惯 → 你立刻知道 | 合作中遇到分歧应该先怎么做 → 需要判断场景 |

---

## 标准三：双方都有对的逻辑

两个选项必须有各自成立的生活逻辑。如果一方看起来"明显更对"，这题就不像在问"你怎么选"，而像是在考"你是什么人"。

选 A 和选 B 的人，都应该觉得自己选得理直气壮。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 火锅更在意锅底 vs 蘸料 → 两派真实存在 | 睡前刷牙 vs 不刷牙 → 一方明显更对 |
| 看电影喜欢独自看 vs 有人一起 → 都有道理 | 坐车系安全带 vs 不系 → 没得选 |

**自查方法**：把这题发到朋友圈，有没有人会真的为两边"吵起来"？有，说明两边的逻辑都站得住。

---

## 标准四：细节够具体

具体场景唤起画面感和身体记忆，让用户不是"思考"回答，而是**感受到**自己的习惯然后点击。抽象的态度题让人无感。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 冬天早晨你更怕：起床那刻 / 出门冷风 | 你是一个怕冷的人吗 |
| 电梯里遇到邻居：主动打招呼 / 点头微笑就好 | 你是一个外向的人吗 |

**画面感原则**：用户读完应该能在脑子里看到那个场景。

---

## 标准五：有被说中的潜力

好题应让人在结果页产生"原来这么多人和我一样"或"原来也有人这样想"的体验。

有两种类型：
- **确认型**：我知道很多人和我一样，但看到比例还是觉得踏实（如"火锅更在意锅底"）
- **惊喜型**：我以为自己是少数，结果发现很多人也一样（如"看电影喜欢独自看"）

不需要每道题都是"惊喜型"，但题库中应有几道有这个潜质。

---

## 标准六：匿名友好（修正版）

> 本标准在团队讨论后已修正。详见下方说明。

在「同频一下」的高度匿名环境中（不留用户记录、无主页、无历史可查），**用户不担心"被看到这一刻"**，因此以下风险不纳入考核：

- ❌ ~~旁人看到屏幕~~ → 这是抖音信息流自带场景，与内容类型无关
- ❌ ~~留下长期记录~~ → 产品设计上已消除

### 真正需要关注的

1. **平台审核风险** — 题面必须能通过抖音内容审核（无性暗示、医疗暗示、违法引导等）
2. **自我形象风险** — 用户不应选完觉得自己是个"坏人"（这和我们是否匿名无关）
3. **行为正常化风险** — 题目不应无意中正常化有害行为

### 修正前后的变化

- **上限变宽**：可纳入情感关系类题目（前任、分手、家庭矛盾等用户真实关心但公开场合不太愿意说的场景）
- **下限不变**：敏感话题、自伤倾向、违法内容仍不进入

| ✅ 可以上（修正后可纳入） | ❌ 仍然不能上 |
|---|---|
| 和前任还能做朋友吗 | 涉及自伤/暴力倾向 |
| 分手后你会删联系方式吗 | 触发敏感词审核 |
| 吵架后你会主动和好吗 | 让用户选完自我归因为"坏人" |
| 你觉得父母更希望你 | 平台安全规则明确禁止的话题 |

## 七种必死题

| 死法 | 例子 | 原因 |
|---|---|---|
| 需要想想 | 工作三年后你更看重薪资还是成长 | 需要权衡，信息流里不会停下 |
| 一边明显对 | 坐车系安全带 vs 不系 | 选了不敢说，不选没意义 |
| 太抽象 | 你相信爱情吗 | 没有画面感，无法一秒回答 |
| 需要知识 | 你更看好新能源还是燃油车 | 不是每个人都有立场 |
| 诱导高低 | 高情商的人会怎么做 | 暗示正确答案，让真实选择者不舒服 |
| 假互斥 | 你更喜欢宅家还是出去玩 | 很多人两者都喜欢，无法一秒站队 |
| 对谁都一样 | 你更喜欢晴天还是阴天 | 太弱，选了也没有"被代表"的感觉 |

## 扩题规则（强制执行）

以下规则补充六条标准，是题库设计、评审、扩题时必须遵守的约束条件。

### 规则一：每小主题只能有 1 题

**必须**保证每个场景/小主题在题库中只出现一次。

| 违反示例 | 说明 |
|---|---|
| "周末早晨，你通常先" + "周末早晨，你的节奏更像" + "周末早晨，有两件事你先" | 同一场景出现 3 题，违反规则 |
| "吃火锅时口味上更爱" + "吃火锅时你会把最爱" + "吃火锅时温度上更爱" | 同一场景出现 3 题，违反规则 |

**例外**：如果两个问题属于完全不同的角度且不会让用户感到重复，最多 2 题。但需特别评审。

### 规则二：禁止模板化批量生成

**禁止**以下行为：
- 一套固定模板（如"你通常先""你的节奏更像""有小事要做你会"）在多个场景间复制粘贴
- 批量替换场景名（如"周末早晨→工作日晚上→刚到家后"）生成相似题目
- 用同一组选项（如"休息一下/把事做完"）在不同场景下重复出现

模板化题目的直接后果是：用户在刷到第 3 道相同结构的题时就会产生"又是这套"的疲劳感，流失率飙升。

**正确做法**：每道题独立设计，题面和选项组合在题库中不应出现重复结构。

### 规则三：情感关系类题目应占最大比例

题库中**情感关系类题目必须占 40% 以上**。原因：
- 情感关系类话题在真实社交平台（豆瓣、贴吧等）讨论热度最高
- 情感类题目代入感强，用户愿意停下来点
- 符合"同频一下"的产品定位——让人感受到"原来不只我这样"

其他类别的建议占比上限：
| 类别 | 建议上限 |
|---|---|
| 生活习惯 | ≤ 15% |
| 饮食偏好 | ≤ 10% |
| 出行习惯 | ≤ 8% |
| 社交倾向 | ≤ 12% |
| 情绪感受 | ≤ 12% |
| 个人偏好 | ≤ 10% |

### 规则四：题目必须来自真实人生困境或日常纠结

**禁止**凭空构思"听起来有道理"的题目。每道题都应能追溯到真实生活中人们确实会纠结、会讨论、会站队的场景。

**合格的题目来源**：
- 豆瓣小组热门讨论（劝分组、情侣观察、人生问题研究社等）
- 百度贴吧情感/生活类板块的争议话题
- 身边真实的人际冲突、习惯差异
- 社交媒体上的热门二选一投票

对每道题应问自己：**"这句话有没有可能出现在真实对话里？"** 如果答案是"不太会有人这么说"，那这道题就不合格。

### 规则五：不得从"例子"反向设计题目

好题标准中的"正例/反例"只用于说明规则，**不是题目的来源或思路**。常见的错误做法：

- 看到"冬天早晨你更怕：起床那刻/出门冷风"是好例子，就去写"夏天下午你更怕：出门那刻/回家路上"——这是错误的，因为你在复用例子的框架
- 看到"电梯里遇到邻居：主动打招呼/点头微笑就好"是好例子，就去写"走廊里遇到邻居：先说话/低头走过去"——同样是错误做法

**正确做法**：从真实生活中找到另一个完全不同的纠结场景，用自己的框架来表达。

### 规则六：每道题必须有独立的画面感

不能有两道题让用户产生"这题和刚才那题差不多"的感觉。检查方法：
- 随机抽两道同类别题目，看它们的前缀/场景/选项结构是否相似
- 如果用户可以给两道题起同一个概括性标题（如"都关于主动性""都关于先做哪个"），说明场景重叠

### 规则七：每次扩题前必须扫描去重

每次新增或修改题目前，**必须**对当前题库做完整扫描，检查以下维度是否有重复：
1. **场景前缀** — 同一场景不能出现两次
2. **选项结构** — 相似的选项框架（如"先A还是先B"）不应在多题中雷同
3. **核心矛盾** — 相同的底层二元对立（如"主动vs被动"）不应反复出现
4. **类别分布** — 新增后情感关系类仍须保持最高占比

**操作方法**：用脚本对比 question_text 和 option_a/option_b 的相似度，输出重复报告，确认无重复后再提交新题。

## 附录：推荐选题方向（按共鸣潜力排序）

> ⚠️ 以下方向仅供参考，每道题必须独立设计，不得做模板化扩展。

| 方向 | 说明 | 典型场景 |
|---|---|---|
| 情感关系 | 人人都会遇到的纠结，代入感强——**应占题库最大比例** | 前任、吵架、分手、家庭期望、催婚、信任、付出与回报 |
| 日常微习惯 | 人人都有自己的做法，容易产生"原来不只我这样" | 吃西瓜、收拾房间、早上醒来的第一个动作 |
| 社交场景 | 内向/外向的分水岭，共识度高 | 电梯遇到邻居、聚会角色、收到消息 |
| 感官偏好 | 有身体记忆，决策快 | 怕冷怕热、下雨天的感受 |
| 消费决策 | 小选择里的个人逻辑 | 旅行准备、换新手机 |
| 代际差异 | 年轻人有共鸣，且有表达欲 | 父母期望、过年回家 |

### 选题来源推荐（持续更新）

以下平台可定期收集题目灵感，但**每个灵感都必须转化为独立题目**，不得做模板扩展：

- 豆瓣：劝分小组、人间情侣观察、人生问题研究社、回避型依恋组
- 百度贴吧：情感吧、恋爱吧、生活吧
- 微博：情感类热搜话题、投票区
- 小红书：情感/生活类争议话题
