# 好题标准｜同频一下题库设计准则

> 文档状态：定稿  
> 依据：PRD-同频一下.md + 团队讨论（2026-07）  
> 用途：题库设计、评审、扩题时的共同判断框架

---

## 核心原则

好的题目不是"有道理"的题目，而是**用户在抖音信息流中刷到时愿意停下来点一下**的题目。

以下六条标准是必选项，不是加分项。新题若有一条不满足，直接淘汰。

---

## 标准一：一秒题感

用户扫一眼就该知道这题在问什么、两个选项各自代表什么。不需要读第二遍，不需要断句。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 周末早上醒来你会 | 周末你更倾向于选择哪种状态度过早晨时光 |
| 吃火锅你更在意 | 在火锅用餐体验中你最看重的因素是什么 |

**判断方法**：一个初中生水平的人，在 0.5 秒内能说出这题是问什么的。

---

## 标准二：自己有答案

用户在读到选项之前，心里已经有了答案。这题不是在帮用户"做决策"，而是在让用户**确认自己已有的偏好**。

如果用户需要权衡利弊、回忆数据、查资料才能回答，就不会在信息流里停下来。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 吃西瓜你更喜欢 → 你立刻知道 | 你认为哪种投资方式更适合年轻人 → 需要想 |
| 收到消息你习惯 → 你立刻知道 | 合作中遇到分歧应该先怎么做 → 需要判断场景 |

---

## 标准三：双方都有对的逻辑

两个选项必须有各自成立的生活逻辑。如果一方看起来"明显更对"，这题就不像在问"你怎么选"，而像是在考"你是什么人"。

选 A 和选 B 的人，都应该觉得自己选得理直气壮。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 火锅更在意锅底 vs 蘸料 → 两派真实存在 | 睡前刷牙 vs 不刷牙 → 一方明显更对 |
| 看电影喜欢独自看 vs 有人一起 → 都有道理 | 坐车系安全带 vs 不系 → 没得选 |

**自查方法**：把这题发到朋友圈，有没有人会真的为两边"吵起来"？有，说明两边的逻辑都站得住。

---

## 标准四：细节够具体

具体场景唤起画面感和身体记忆，让用户不是"思考"回答，而是**感受到**自己的习惯然后点击。抽象的态度题让人无感。

| ✅ 通过 | ❌ 不通过 |
|---|---|
| 冬天早晨你更怕：起床那刻 / 出门冷风 | 你是一个怕冷的人吗 |
| 电梯里遇到邻居：主动打招呼 / 点头微笑就好 | 你是一个外向的人吗 |

**画面感原则**：用户读完应该能在脑子里看到那个场景。

---

## 标准五：有被说中的潜力

好题应让人在结果页产生"原来这么多人和我一样"或"原来也有人这样想"的体验。

有两种类型：
- **确认型**：我知道很多人和我一样，但看到比例还是觉得踏实（如"火锅更在意锅底"）
- **惊喜型**：我以为自己是少数，结果发现很多人也一样（如"看电影喜欢独自看"）

不需要每道题都是"惊喜型"，但题库中应有几道有这个潜质。

---

## 标准六：匿名友好（修正版）

> 本标准在团队讨论后已修正。详见下方说明。

在「同频一下」的高度匿名环境中（不留用户记录、无主页、无历史可查），**用户不担心"被看到这一刻"**，因此以下风险不纳入考核：

- ❌ ~~旁人看到屏幕~~ → 这是抖音信息流自带场景，与内容类型无关
- ❌ ~~留下长期记录~~ → 产品设计上已消除

### 真正需要关注的

1. **平台审核风险** — 题面必须能通过抖音内容审核（无性暗示、医疗暗示、违法引导等）
2. **自我形象风险** — 用户不应选完觉得自己是个"坏人"（这和我们是否匿名无关）
3. **行为正常化风险** — 题目不应无意中正常化有害行为

### 修正前后的变化

- **上限变宽**：可纳入情感关系类题目（前任、分手、家庭矛盾等用户真实关心但公开场合不太愿意说的场景）
- **下限不变**：敏感话题、自伤倾向、违法内容仍不进入

| ✅ 可以上（修正后可纳入） | ❌ 仍然不能上 |
|---|---|
| 和前任还能做朋友吗 | 涉及自伤/暴力倾向 |
| 分手后你会删联系方式吗 | 触发敏感词审核 |
| 吵架后你会主动和好吗 | 让用户选完自我归因为"坏人" |
| 你觉得父母更希望你 | 平台安全规则明确禁止的话题 |

## 七种必死题

| 死法 | 例子 | 原因 |
|---|---|---|
| 需要想想 | 工作三年后你更看重薪资还是成长 | 需要权衡，信息流里不会停下 |
| 一边明显对 | 坐车系安全带 vs 不系 | 选了不敢说，不选没意义 |
| 太抽象 | 你相信爱情吗 | 没有画面感，无法一秒回答 |
| 需要知识 | 你更看好新能源还是燃油车 | 不是每个人都有立场 |
| 诱导高低 | 高情商的人会怎么做 | 暗示正确答案，让真实选择者不舒服 |
| 假互斥 | 你更喜欢宅家还是出去玩 | 很多人两者都喜欢，无法一秒站队 |
| 对谁都一样 | 你更喜欢晴天还是阴天 | 太弱，选了也没有"被代表"的感觉 |

## 附录：推荐选题方向（按共鸣潜力排序）

| 方向 | 说明 | 举例 |
|---|---|---|
| 日常微习惯 | 人人都有自己的做法，容易产生"原来不只我这样" | 吃西瓜、洗澡顺序、收拾房间 |
| 社交场景 | 内向/外向的分水岭，共识度高 | 电梯遇到邻居、聚会角色、收到消息 |
| 情感关系 | 人人都会遇到的纠结，代入感强 | 前任、吵架、分手、家庭期望 |
| 感官偏好 | 有身体记忆，决策快 | 怕冷怕热、下雨天的感受 |
| 消费决策 | 小选择里的个人逻辑 | 旅行准备、换新手机 |
| 代际差异 | 年轻人有共鸣，且有表达欲 | 父母期望、过年回家 |
